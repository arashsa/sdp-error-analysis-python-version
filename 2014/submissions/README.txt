
SemEval 2014 Task 8: Broad-Coverage Semantic Dependency Parsing: Final Results

Version 1.0; April 22, 2014


Overview
========

This directory contains system submissions (i.e. results on the task-specific
test data) from nine participating teams.  Each directory follows the layout
and data format described on the task web pages:

  http://sdp.delph-in.net/2014/participation.html
  http://sdp.delph-in.net/2014/data.html

For each submission, the output of the official scorer (in its version of April
16, 2014) is included in each directory, with the common file suffix ‘.scores’.
